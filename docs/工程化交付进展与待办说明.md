## 2. 当前有效代码量统计

### 2.1 统计口径

本节代码量统计遵循以下约定：

- **统计范围（分层）**
  - Go 源码：`api/`、`cmd/`、`internal/`
  - Shell 脚本：`scripts/`
  - Helm-YAML：`deploy/helm/`
  - 其他工程文件：`Makefile`（仅此一项，无 `Dockerfile`、`build/`、`config/`）
- **排除范围**：`vendor/`、`.git/`、`.worktrees/`、`artifacts/`、`docs/`、bundle 产物
- **"有效代码行"定义**：排除空行与纯注释行（`//`、`#`、`*` 开头）后的近似可执行/可维护代码量，用于工程规模感知，不追求审计级精确值

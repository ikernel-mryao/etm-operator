以下为三份文档共享的已验证事实基线：

- control-plane loop proven
- agent/DaemonSet/node execution loop proven
- data-plane loop proven by VmSwap > 0 and Swap used growth
- delivery chain proven by Helm install/upgrade, offline image import, workload patching, script/NodeState observation
